1. Fork and git clone this repo
2. Download "Docker Desktop" on your local computer and run it
3. Open up VSCode and download the extension called "Dev Containers"
4. Open up the cloned repo in VSCode. When you do this you should be prompted at the bottom right of VSCode with a message asking if you want to reopen this in a container

Note: You can get different containers for different versions of python and AVD here:
https://github.com/aristanetworks/avd/pkgs/container/avd%2Funiversal/versions?filters%5Bversion_type%5D=tagged
